# Unicampi Browser

Um frontend em React/Redux

# Demo

[https://unicampi.github.io/unicampi-browser/dist/#/](https://unicampi.github.io/unicampi-browser/dist/#/)


# Pré-Requisitos

Estou usando o [Yarn](https://yarnpkg.com/) no lugar do NPM

# Instalação

    $ cd <path to project>/
    $ yarn

# Rodando

Ambiente de DEV

    $ yarn run start-dev

Build de PROD

    $ yarn run build
