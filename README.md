# Unicampi Browser

A frontend for [Unicampi API](https://github.com/gabisurita/unicampi)


# Pré-Requisitos

Estou usando o [Yarn](https://yarnpkg.com/) no lugar do NPM

# Instalação

    $ cd <path to project>/
    $ yarn

# Rodando

Ambiente de DEV

    $ yarn run start-dev

Build de PROD

    $ yarn run build
